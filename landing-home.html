Great developer experience

Functional mode

With 'functional mode' you can write cleaner, shorter code that doesn't rely on your IDE as a crutch. Functional code, still uses namespaces in the background, so if there are any class naming collisions it can still be addressed. All it does it make your code shorter and easier to read. If you prefer you can still use a traditional OOP approach.
