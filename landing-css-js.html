Shallow learning curve

Step one

If you're familiar with bootstrap's grid system and tailwind, you will be right at home with our CSS framework. Simply include 'main.css' in your app and you can start using it right away.

Step two

For the sake of sanity, we've created only a limited number of Vuejs components. These do NOT require a build step. Just include the relevant script tags wherever you need them.

Step three

You're done! That's all there is to it in a nutshell. With this workflow you can create dynamic websites. You might not have the fancy single page application work, but the small page refreshes required compensates for the complicated client side routing setup.
